# Track B MIA

Manager-Planner-Executor style exploration on top of the shared SII tools. The first implementation provides task classification, static planning, workflow memory, and no-gold reviewer scaffolding.

References:

- Memory Intelligence Agent (MIA): Manager-Planner-Executor and Reviewer-AC inspiration.
- Agent Workflow Memory: reusable workflow memory from trajectories.
- ExpeL: extracting natural-language experience from trajectories.

Benchmark mode disables online memory updates and parameter updates.

References:

- SAGE: Multi-Agent Self-Evolution for LLM Reasoning.
- RAGEN: Understanding Self-Evolution in LLM Agents via Multi-Turn Reinforcement Learning, for multi-turn agent training motivation.
