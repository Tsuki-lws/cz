# Track C AHE

Agentic Harness Engineering exploration. The first implementation loads componentized prompts, runs the shared ReAct harness, and records a manifest for later evolution on dev data.

References:

- GEPA: reflective prompt evolution from trajectories.
- Training LLM Agents for Spontaneous, Reward-Free Self-Evolution via World Knowledge Exploration: world-knowledge exploration and summarization.
- Voyager: reusable skill-library framing.
- Agent-Pro: policy-level reflection.

Benchmark mode disables manifest/prompt/memory evolution updates.

References:

- Agentic Harness Engineering: Observability-Driven Automatic Evolution of Coding-Agent Harnesses (arXiv:2604.25850).
- Training LLM Agents for Spontaneous, Reward-Free Self-Evolution via World Knowledge Exploration (arXiv:2604.18131). We only adopt the fixed `world_knowledge.md` idea for one-day feasibility; full SFT/RFT is out of scope.
